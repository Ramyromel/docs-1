
# Smart Contract Addresses

The ERC-6551 registry and Tokenbound account contracts are permissionlessly deployable at the following addresses on any chain.

If the chain you are building on is not in the table below, you can deploy the contracts yourself using our [self-deployment tool](https://tokenbound-v3-deployer.vercel.app/).
